Blood Donation Insights Dashboard
Project Overview

This Power BI dashboard provides insights into blood donation activities, donor distribution, blood group availability, and donation trends over time.

The dashboard helps healthcare organizations and blood banks monitor donation patterns and donor availability for better planning and decision-making.
